# 安装包

## 操作步骤

1. 点击Window -> Show View -> Other -> RuyiSDK -> Ruyi Package Explorer进入包管理器。
3. 选择需要安装的包后点击 Download。
4. 命令行和视图同步查看安装情况。

## 预期结果
1. 成功完成安装，输出 `RUYI_DONE`。

2. 命令行执行 `~/.local/bin/ruyi list --is-installed 1`，显示Install的包。
