### 配置好必要的环境

```
sudo dnf install -y gcc make ncurses-devel git
```

### 开始

- 获取项目

```
git clone https://github.com/mevdschee/2048.c.git
```

- 进入项目

```
cd 2048.c
```

- 针对 RISC-V 编译

```
gcc -o 2048_rv 2048.c -lncurses
```

- 开始游玩

```
./2048_rv
```
